A&P ePortfolio Static Site

How to run (no coding tools needed):
1) Unzip the folder.
2) Double-click index.html to open in your browser.

If your teacher wants a web link:
- Upload the folder to Google Drive + use a simple hosting option (GitHub Pages is common),
  or ask your instructor which hosting is allowed.

Where to edit:
- Each body system page is its own .html file.
- Styles are in styles.css.
- Tiny JS is in main.js.

Add images:
- Put images in /assets/
- Update the <img src="assets/your-image.ext"> paths.
